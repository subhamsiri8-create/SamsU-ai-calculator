Medical Coding 101

ICD-10-CM (Diagnosis)

Used to describe WHAT is wrong with the patient. (e.g., Flu, Broken Bone, Heart Disease).

Format: 3-7 characters (e.g., J10.1)

CPT (Procedures)

Used to describe WHAT WAS DONE to fix it. (e.g., Surgery, X-Ray, Office Visit).

Format: 5 numbers (e.g., 99213)

HCPCS (Equipment/Drugs)

Used for supplies like wheelchairs, ambulances, or specific drugs.
